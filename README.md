# VogueVista

A premium fashion e-commerce platform designed with an "Obsidian & Gold" aesthetic.

## Features

- **Premium Design**: Sleek "Obsidian & Gold" theme providing a luxurious user experience.
- **Dynamic Navigation**: Filter products by Men, Women, and specific categories.
- **Interactive UI**: Smooth animations, hover effects, and a responsive layout.
- **Shopping Cart**: Functional cart with add/remove capabilities.
- **Product Showcase**: extensive product catalog with high-quality images.

## Technology Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom properties for theming)

## Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
